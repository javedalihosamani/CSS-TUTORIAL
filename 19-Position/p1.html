CSS POSITION..!

Positioning elements appropriately on the web pages is a necessity for a good layout design.

There are 4 values

  1. RELATIVE - Current position as ref
  2. ABSOLUTE - Body as a ref and scroll along with content
  3. FIXED - Body as a ref , will not scroll along with content
  4. STICKY - Depend on scrolling position

Position 4 properties

  1. TOP
  2. RIGHT
  3. BOTTOM
  4. LEFT
